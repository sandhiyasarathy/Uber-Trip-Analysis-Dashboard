# Uber Trip Analysis Dashboard - Power BI Project

## Project Overview
This project is a comprehensive Power BI dashboard created to analyze Uber ride data for the month of June 2024 in New York City. The dashboard provides detailed insights into booking patterns, trip distances, payment methods, vehicle usage, and location analytics to help understand Uber ride trends and business performance.

## Key Features
- Total bookings and booking value analysis
- Average trip distance and trip time metrics
- Payment type distribution and analysis
- Vehicle type usage overview
- Time series analysis showing booking trends by hour and day
- Location insights showing frequent pickup and drop-off points
- Interactive filters, drill-throughs, and bookmarks for enhanced exploration

## Data Sources
- Uber trip details dataset containing over 100,000 records
- Location dataset with pickup and drop-off points

## How to Use
1. Open the Power BI Desktop and load the `Uber_Trip_Analysis.pbix` file.
2. Explore the interactive dashboard using slicers, charts, and KPIs.
3. Review the drill-through pages for deeper insights into specific segments.
4. Use the bookmarks to navigate between different dashboard views.

## Requirements
- Power BI Desktop (free download from Microsoft)
- Basic familiarity with Power BI and data visualization concepts

## Notes
- The data used is for demonstration purposes only and represents real-world scenarios.
- The "Publish to web" feature can be used to share the dashboard online but requires a Power BI account.
